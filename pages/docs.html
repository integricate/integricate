Introduction to Integricate

Integricate is a comprehensive UAS (Unmanned Aerial Systems) programming platform that enables developers to build, test, and deploy autonomous drone applications with ease.

What is Integricate?

Integricate provides a unified API for controlling drones, managing missions, processing telemetry data, and integrating computer vision capabilities. Whether you're building a single drone application or managing a fleet of hundreds, Integricate scales with your needs.

Getting Started

Quick Start Guide

Get your first drone flying with Integricate in under 5 minutes.

Prerequisites

  • Python 3.8+ or C++17
  • A MAVLink-compatible drone or SITL simulator
  • An Integricate API key (Get one free)

Step 1: Install the SDK

bash 
# Install the Python SDK
pip install integricate

# Or install with extras for computer vision
pip install integricate[vision]

Step 2: Configure Authentication

Set your API key as an environment variable:

bash 
export INTEGRICATE_API_KEY="your-api-key-here"

Step 3: Connect to Your Drone

python 
from integricate import Drone

# Auto-detect and connect to your drone
drone = Drone.connect("auto")

# Or specify connection explicitly
drone = Drone.connect("serial:///dev/ttyUSB0:57600")
drone = Drone.connect("udp://:14540")  # For SITL

# Check connection status
print(f"Connected to: {drone.system_id}")
print(f"Firmware: {drone.firmware_version}")

Step 4: Take Off!

python 
import asyncio
from integricate import Drone

async def main():
    # Connect to drone
    drone = Drone.connect("auto")

    # Pre-flight checks
    if not drone.is_ready:
        print("Drone not ready. Check:")
        print(drone.preflight_status)
        return

    # Arm and takeoff
    await drone.arm()
    await drone.takeoff(altitude=10)  # 10 meters

    # Hover for 5 seconds
    await asyncio.sleep(5)

    # Land safely
    await drone.land()

    print("Mission complete!")

asyncio.run(main())

Installation

Detailed installation instructions for all supported platforms and languages.

Python SDK Installation

The Python SDK requires Python 3.8 or later.

Using pip (Recommended)

bash 
# Basic installation
pip install integricate

# With computer vision support
pip install integricate[vision]

# With all optional dependencies
pip install integricate[all]

From Source

bash 
git clone https://github.com/integricate/integricate-python.git
cd integricate-python
pip install -e .

C++ SDK Installation

The C++ SDK requires C++17 and CMake 3.16+.

Using CMake

cmake 
include(FetchContent)
FetchContent_Declare(
  integricate
  GIT_REPOSITORY https://github.com/integricate/integricate-cpp.git
  GIT_TAG v3.0.0
)
FetchContent_MakeAvailable(integricate)

target_link_libraries(your_target PRIVATE integricate::integricate)

ROS 2 Integration

The ROS 2 package works with Humble and Iron.

Using rosdep

bash 
# Add to your workspace src folder
cd ~/ros2_ws/src
git clone https://github.com/integricate/integricate_ros.git

# Install dependencies
rosdep install --from-paths src --ignore-src -r -y

# Build
cd ~/ros2_ws
colcon build --packages-select integricate_ros
API Reference

Drone API

Core API endpoints for drone control and status.

Endpoints

POST /api/v1/drone/arm

Arm the drone motors. Requires safety checks to pass.

Request Body

{
  "drone_id": "drone-alpha-01",
  "force": false
}

Response

{
  "success": true,
  "drone_id": "drone-alpha-01",
  "armed": true,
  "timestamp": "2024-01-15T14:32:18Z"
}
POST /api/v1/drone/takeoff

Command the drone to take off to a specified altitude.

Request Body

{
  "drone_id": "drone-alpha-01",
  "altitude": 30,
  "speed": 5,
  "heading": null
}

Parameters

Parameter Type Required Description
drone_id string Yes Unique identifier of the drone
altitude number Yes Target altitude in meters (AGL)
speed number No Climb rate in m/s (default: 3)
heading number No Target heading in degrees (null = maintain current)
GET /api/v1/drone/telemetry

Get real-time telemetry data from the drone.

Query Parameters

Parameter Type Required Description
drone_id string Yes Unique identifier of the drone
include string No Comma-separated list: gps, battery, imu, attitude

Response

{
  "drone_id": "drone-alpha-01",
  "timestamp": "2024-01-15T14:32:18Z",
  "gps": {
    "latitude": 47.3977,

  },
