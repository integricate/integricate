# Integricate - UAS Programming Platform

A modern, responsive website for Integricate - a comprehensive UAS (Unmanned Aerial Systems) programming platform for drone development.

## Overview

Integricate is designed to help developers build, test, and deploy autonomous drone applications. This website showcases the platform's features, API capabilities, pricing plans, and documentation.

## Features

- **Modern Design**: Clean, professional UI inspired by leading tech platforms
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**: Code tabs, API playground, smooth animations
- **Comprehensive Pages**: Home, Documentation, and Pricing

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox, animations
- **JavaScript** - Vanilla JS for interactivity
- **Google Fonts** - Inter typeface

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - pure HTML/CSS/JS

## Design Highlights

### Hero Section
- Animated gradient background with floating orbs

- Live statistics counter animations

### Features Section
- Six-card grid layout with featured card spanning
- Icon-based feature presentation
- Hover animations and transitions

### API Demo
- Interactive endpoint selector
- Request/response panels
- Simulated API calls with animations

### Documentation
- Three-column layout with sidebar navigation

- Code blocks with copy functionality
- Tabbed content for multiple SDKs

### Pricing
- Monthly/yearly toggle

- FAQ accordion section
